# GearMind: ระบบประมวลผลภาษาธรรมชาติ

ขก.เขียนโว้ยยยยยย

## เริ่มต้นใช้งาน

### การติดตั้ง

```
pnpm i
```

### การรวม data มาเป็น essay

```
node md-to-txt.js
```

### การพัฒนา

```
pnpm dev
```

### การ Build

```
pnpm build
```

## โครงสร้างโฟลเดอร์

```
เอาไว้ก่อนๆ เดี๋ยวมาเขียน
```
